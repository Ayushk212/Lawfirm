# Vadhera & Associates

A premium, modern, and production-ready web application for **Vadhera & Associates**—a full-service law firm providing exceptional advocacy and counsel since 1998.

## Features & Pages

- **Home / Hero**: Cinematic landing layout featuring active animations, firm introduction, and highlights of core practice groups.
- **Our Firm (About)**: Chronological firm history, milestone timeline, corporate culture, and ethical philosophy.
- **Practice Areas**: Detailed breakdown of Litigation, Corporate Counsel, and Estate Planning with client intakes.
- **Attorneys**: Partner and associate profile cards outlining education, credentials, certifications, and consultation direct mail hooks.
- **Contact**: Full interactive case evaluation intake form with real-time field validation, operating schedules, and map location panels.

## Technology Stack

- **Framework**: React 19 + TypeScript
- **Styling**: Tailwind CSS v4 + custom HSL color tokens (Deep Navy `#0A1C35`, Cream `#F5F2EA`, Antique Gold `#B9975B`)
- **Routing**: React Router DOM
- **Animations**: Framer Motion (via `motion/react`)
- **Icons**: Lucide React

## Running Locally

1. **Install Dependencies**:
   ```bash
   npm install
   ```
2. **Start the Development Server**:
   ```bash
   npm run dev
   ```
3. **Build for Production**:
   ```bash
   npm run build
   ```
